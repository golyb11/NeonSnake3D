# Инструкция по запуску и остановке сервера для игры "Змейка"

Перед отправкой сборки на Яндекс Игры см. файл YANDEX_GAMES_RELEASE.txt (чеклист SDK и ZIP).

Готовый архив для загрузки в Консоль: файл yandex-snake-upload.zip в корне проекта.
Собрать его заново (бандл + zip): из папки игры выполните
  npm run release:yandex

## Как запустить сервер

Для корректной работы игры (особенно если используются модули или текстуры) рекомендуется запускать её через локальный сервер.

### Способ 1: Через Python (самый простой)
Если у вас установлен Python, выполните следующие действия:
1. Откройте терминал (PowerShell или Командная строка) в папке с игрой (`d:\zmeika`).
2. Введите команду:
   ```bash
   python -m http.server 8000
   ```
3. Откройте браузер и перейдите по адресу:
   [http://localhost:8000](http://localhost:8000)

### Способ 2: Через Node.js (если установлен)
1. Откройте терминал в папке с игрой.
2. Введите команду:
   ```bash
   npx serve .
   ```
3. Откройте браузер по адресу, который укажет команда (обычно [http://localhost:3000](http://localhost:3000)).

---

## Как остановить сервер

1. Перейдите в окно терминала, где запущен сервер.
2. Нажмите комбинацию клавиш **Ctrl + C** на клавиатуре.
3. Если появится запрос "Завершить выполнение пакетного файла? (Y/N)", введите `Y` и нажмите **Enter**.
4. Теперь сервер остановлен, и страница в браузере перестанет обновляться.
