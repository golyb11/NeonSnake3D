# Neon Snake 3D

Классическая змейка в полностью переработанном 3D-формате с неоновой эстетикой, эффектами частиц и ракетным ускорением. Подготовлено для публикации на Яндекс.Играх.

## Стек технологий

- HTML5 + CSS3
- JavaScript ES2020 (ES Modules)
- Three.js 0.160 — 3D-графика
- EffectComposer + UnrealBloomPass — постпроцессинг и свечение
- esbuild — бандлинг
- Яндекс.Игры SDK — интеграция с платформой

## Описание функционала

### Управление

| Действие | Клавиатура | Тачскрин |
|----------|-----------|----------|
| Движение | W/A/S/D | Свайпы |
| Ракета (ускорение) | Двойной тап + удержание | Двойной тап + удержание |
| Лазер (взрыв препятствий) | Клик мыши | Касание |

### Особенности

- **3D-геймплей** на светящейся сетке с динамичной камерой
- **Неоновый стиль** с эффектом bloom (свечение объектов)
- **Ракетное ускорение** — временное увеличение скорости
- **Лазер** — уничтожение препятствий на пути
- **Частицы** — неоновый след и эффекты взрывов
- **Адаптивность** — поддержка ПК и мобильных устройств
- **Интеграция с Яндекс.Играми** — рекорды, реклама, полноэкранный режим

### Цель игры

- Собирай светящуюся еду (+10 очков за каждую)
- Уворачивайся от стен и своего хвоста
- Взрывай препятствия лазером
- Ставь рекорды!

## Инструкция по запуску

### Локальный запуск (разработка)

```bash
# Клонирование
git clone <репозиторий>
cd zmeika

# Установка зависимостей
npm install

# Сборка бандла
npm run build

# Запуск локального сервера
python -m http.server 8000
# или
npx serve .

# Открыть в браузере
# http://localhost:8000
```

### Сборка для релиза (Яндекс.Игры)

```bash
npm run release:yandex
```

Готовый архив: `yandex-snake-upload.zip`

## Архитектура игры

### Основные системы

- **Snake** — логика змейки, движение по сетке, рост
- **Food** — генерация еды, визуальные эффекты
- **Obstacles** — препятствия и их уничтожение лазером
- **Boost** — система ускорения (ракета)
- **Laser** — взрыв препятствий по клику
- **Trail** — частицы неонового следа
- **Input** — обработка клавиатуры и тачскрина
- **Yandex SDK** — интеграция с платформой

### Post-processing

- `RenderPass` — рендеринг сцены
- `UnrealBloomPass` — эффект свечения для неоновой эстетики
